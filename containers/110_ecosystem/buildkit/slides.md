## BuildKit

*concurrent, cache-efficient, and Dockerfile-agnostic builder toolkit*

### The project

- Initiated by Docker
- Community driven

### Relevant features

- Multi-stage builds
- Parallel builds
- Remote build cache
- Build mounts
- Build secrets
- SSH forwarding

Working rootless [implementation](https://github.com/moby/buildkit/blob/master/docs/rootless.md)
